## ðŸ”§ Configuration Guide

### Payment Gateway Setup

#### PayPal
1. Create a PayPal Developer account at https://developer.paypal.com
2. Create a new REST API app
3. Copy your Client ID and Secret
4. Update `config.php` with your credentials

#### GCash
1. Register as a GCash merchant
2. Obtain your Merchant ID and API Key
3. Update `config.php` with your credentials
4. Configure webhook URLs for payment notifications

#### Stripe (Credit/Debit Cards)
1. Create a Stripe account at https://stripe.com
2. Get your API keys from the dashboard
3. Update `config.php` with your keys
4. Test with test cards in sandbox mode

### Email Configuration

For Gmail SMTP:
1. Enable 2-Factor Authentication on your Google account
2. Generate an App Password
3. Use this App Password in `SMTP_PASS`
